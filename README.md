# Journal

> Public and Private Journals 
- Add user Specific Journals
- Add Public journals
- view them in according to the user
- edit the journal

## Usage

Add your mongoDB URI and Google OAuth credentials to the config.env file

```
# Install dependencies
npm install

# Run in development
npm run dev

# Run in production
npm start
```
